# Life Tracker - Web Edition

A stunning, professional life management platform built with Vanilla HTML, CSS, and JS.

## Features
- **Dashboard**: High-level overview of daily stats and recent activities.
- **Habit Tracker**: Routine management with streak tracking.
- **Financial Ledger**: Expense and income tracking with balance calculation.
- **Time Clock**: Productivity focus timer with stopwatch functionality.
- **Analytics**: Visual data insights powered by Chart.js.
- **Theme**: Premium Glassmorphism UI with Neon accents.
- **Persistence**: All data is automatically saved to your browser's Local Storage.

## How to Run
Simply open `index.html` in any modern web browser.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Glassmorphism, Animations)
- Vanilla JavaScript (ES6+)
- Chart.js (via CDN for high-end visualizations)
- Font Awesome (via CDN for professional iconography)
- Google Fonts (Outfit & Inter)
